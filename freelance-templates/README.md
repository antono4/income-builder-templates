## 💡 TIPS PENTING

### 3 Kesalahan Fatal Freelancer Pemula:

1. **❌ Kerja tanpa kontrak**
   → Selalu gunakan kontrak! Lindungi diri Anda.

2. **❌ Underpricing**
   → Price based on value, not time. Jangan jual mahal undervalue diri sendiri.

3. **❌ generic proposal**
   → Setiap proposal HARUS dipersonalisasi. Tidak ada yang mau baca template umum.

### 3 Kebiasaan Freelancer Sukses:

1. **✅ Konsisten kirim proposal**
   → Target: 5-10 proposal per hari sampai pipeline penuh.

2. **✅ Deliver excellent work**
   → Lebih baik underpromise dan overdeliver. Bangun reputasi!

3. **✅ Collect reviews**
   → Reviews adalah mata uang freelance. Minta testimonial setelah setiap project.
